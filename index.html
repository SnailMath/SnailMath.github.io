hollyhock-2 is a program loader for the classpad II aka fx-cp400 which enables you to run programs written in assembly and c on your classpad II.

In 2020, I found an article by The6P4C where they modified the firmware of the Casio Classpad II aka fx-cp 400, so they could run their own programs like games on the calculator. I had to find out how they did it, so I wrote my own launcher, the LOADER IIb, and made the first youtube video about it (see below).

After the video was pubished, I found many other people, who wanted to play games and program for the calculator, so we started the inofficial Classpad Discord Server. With so many people interested in the calculator, we quickly found the original hollyhock launcher by The6P4C, so we immediately abandoned my simple LOADER IIb and continiued work on hollyhock, making it hollyhock-2, so we have all the same foundation for all our games and programs.

(The only difference between hollyhock and hollyhock-2 is that hollyhock-2 is much easier to insatll, it preserves the exam mode and it adds a simple .bin file type and the ability to write programs in machine code .)
